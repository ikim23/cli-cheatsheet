# VSCode

- https://vscode-shortcuts.com/

| Command              | Shortcut                                          |
| -------------------- | ------------------------------------------------- |
| Show command palette | <kbd>Shift</kbd> + <kbd>Ctrl</kbd> + <kbd>P</kbd> |
| Go to file           | <kbd>Ctrl</kbd> + <kbd>P</kbd>                    |
| Go forward           | <kbd>Ctrl</kbd> + <kbd>-</kbd>                    |
| Go backward          | <kbd>Shift</kbd> + <kbd>Ctrl</kbd> + <kbd>-</kbd> |
