# VSCode

- https://vscode-shortcuts.com/
- https://code.visualstudio.com/docs/getstarted/keybindings#_basic-editing

| Command                                     | Shortcut                                                    |
| ------------------------------------------- | ----------------------------------------------------------- |
| Show All Commands                           | <kbd>cmd</kbd> + <kbd>shift</kbd> + <kbd>P</kbd>            |
| Go to File                                  | <kbd>cmd</kbd> + <kbd>P</kbd>                               |
| Delete Line                                 | <kbd>shift</kbd> + <kbd>cmd</kbd> + <kbd>K</kbd>            |
| Undo                                        | <kbd>cmd</kbd> + <kbd>Z</kbd>                               |
| Redo                                        | <kbd>shift</kbd> + <kbd>cmd</kbd> + <kbd>Z</kbd>            |
| Move Line Up                                | <kbd>option</kbd> + <kbd>↑</kbd>                            |
| Move Line Down                              | <kbd>option</kbd> + <kbd>↓</kbd>                            |
| Copy Line Up                                | <kbd>shift</kbd> + <kbd>option</kbd> + <kbd>↑</kbd>         |
| Copy Line Down                              | <kbd>shift</kbd> + <kbd>option</kbd> + <kbd>↓</kbd>         |
| Go Back                                     | <kbd>ctrl</kbd> + <kbd>-</kbd>                              |
| Go Forward                                  | <kbd>ctrl</kbd> + <kbd>shift</kbd> + <kbd>-</kbd>           |
| Find                                        | <kbd>cmd</kbd> + <kbd>F</kbd>                               |
| Replace                                     | <kbd>option</kbd> + <kbd>cmd</kbd> + <kbd>F</kbd>           |
| Select all occurrences of current selection | <kbd>shift</kbd> + <kbd>cmd</kbd> + <kbd>L</kbd>            |
| Rename Symbol                               | <kbd>F2</kbd>                                               |
| Quick Fix                                   | <kbd>cmd</kbd> + <kbd>.</kbd>                               |
| Show Hover                                  | <kbd>cmd</kbd> + <kbd>K</kbd> <kbd>cmd</kbd> + <kbd>I</kbd> |
| Go to Definition                            | <kbd>F12</kbd>                                              |
| Close Editor                                | <kbd>cmd</kbd> + <kbd>W</kbd>                               |
| Close All Editors                           | <kbd>cmd</kbd> + <kbd>K</kbd> <kbd>cmd</kbd> + <kbd>W</kbd> |
| Split Editor                                | <kbd>cmd</kbd> + <kbd>\</kbd>                               |
| Focus into First Editor Group               | <kbd>cmd</kbd> + <kbd>1</kbd>                               |
